# Chapter 2: Explicit Typing and Inference

## What you learn
- Type inference (TypeScript guesses types)
- Explicit typing (you define types)
- `as const` for literal precision

## Key idea
Use inference by default for cleaner code, and add explicit types when clarity or API boundaries need it.

## Run
```bash
npm run build
node dist/Explicit&Inference/ExplicitAndInference.js
```

## Practice
- Add a `salary` variable using explicit type.
- Add a function with inferred return type.
- Try assigning `"user"` to `currentRole` and observe compile error.
