# Chapter 13: Advanced Types

## What you learn
- Mapped types
- Conditional types
- `infer` keyword
- Template literal types

## Why this matters
Advanced types help you model APIs and reusable internal frameworks with precision.

## Run
```bash
npm run build
node dist/advancedTypes/AdvancedTypes.js
```

## Practice
- Build a type that makes all nested fields optional.
- Create route types like `"PATCH /users/:id"`.
- Use conditional types to unwrap array element types.
