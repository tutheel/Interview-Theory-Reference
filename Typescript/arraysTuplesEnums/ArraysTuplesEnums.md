# Chapter 5: Arrays, Tuples, and Enums

## What you learn
- Typed arrays (`string[]`, `number[]`)
- Tuples for fixed-position data
- Enums for named constants

## Why this matters
These patterns model lists, coordinate-like data, and state values clearly.

## Run
```bash
npm run build
node dist/arraysTuplesEnums/ArraysTuplesEnums.js
```

## Practice
- Create a tuple for `[username, isActive]`.
- Add one more enum member and print it.
- Build an array of tuples representing users.
