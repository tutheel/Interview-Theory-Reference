# Chapter 10: Classes and OOP

## What you learn
- Classes and constructors
- Access modifiers (`public`, `private`)
- Inheritance
- Abstract classes and methods

## Why this matters
TypeScript makes OOP models safe and maintainable with typed contracts.

## Run
```bash
npm run build
node dist/classesOOP/ClassesOOP.js
```

## Practice
- Add a `Manager` class extending `Employee`.
- Add `protected` data and expose it via methods.
- Add readonly properties to your models.
