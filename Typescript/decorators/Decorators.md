# Chapter 16: Decorators

## What you learn
- Method decorators
- Intercepting method calls
- Logging cross-cutting behavior

## Why this matters
Decorators are useful for concerns like logging, metrics, and authorization.

## Run
```bash
npm run build
node dist/decorators/Decorators.js
```

## Note
Decorators are enabled in this project via `experimentalDecorators: true`.

## Practice
- Add a timing decorator.
- Add a validation decorator for input ranges.
- Try class-level decorator examples.
