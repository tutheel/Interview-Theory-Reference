# Chapter 3: Functions

## What you learn
- Parameter typing
- Optional parameters (`?`)
- Return types
- Arrow functions
- Rest parameters
- Function type aliases

## Why this matters
Functions are your strongest type boundary. Good function signatures prevent many bugs.

## Run
```bash
npm run build
node dist/functions/Functions.js
```

## Practice
- Create a function with default parameter values.
- Create a function type alias for string formatters.
- Add a function returning `void` and one returning `never`.
