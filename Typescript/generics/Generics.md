# Chapter 9: Generics

## What you learn
- Generic functions
- Generic interfaces
- Generic classes
- Constrained generics (`K extends keyof T`)

## Why this matters
Generics let you keep type safety while writing reusable code.

## Run
```bash
npm run build
node dist/generics/Generics.js
```

## Practice
- Build a generic `mapArray<T, U>` function.
- Create a generic cache class.
- Create a generic API error wrapper.
