# Chapter 11: Modules and Namespaces

## What you learn
- Legacy `namespace` usage
- Module-style structuring with exported functions/objects
- Organizing code boundaries

## Why this matters
Modern TypeScript projects use ES modules, but namespace knowledge helps with older codebases.

## Run
```bash
npm run build
node dist/modulesNamespaces/ModulesNamespaces.js
```
