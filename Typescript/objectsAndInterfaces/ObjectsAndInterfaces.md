# Chapter 4: Objects and Interfaces

## What you learn
- Defining object shapes with `interface`
- Optional properties (`?`)
- Readonly properties
- Nested object typing

## Why this matters
Interfaces make complex data structures predictable and readable.

## Run
```bash
npm run build
node dist/objectsAndInterfaces/ObjectsAndInterfaces.js
```

## Practice
- Add a `phone?: string` field to `User`.
- Create another interface `Admin` extending `User`.
- Build a function that accepts an array of `User` and prints names.
