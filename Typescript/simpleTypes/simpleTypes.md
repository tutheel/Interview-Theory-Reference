# Chapter 1: Simple Types

## What you learn
- Primitive types: `string`, `number`, `boolean`
- Collection basics: arrays (`number[]`)
- Special values: `null`
- Safer dynamic values with `unknown`

## Why this matters
Simple types are the foundation of TypeScript. They help you catch mismatches at compile time instead of runtime.

## Run
```bash
npm run build
node dist/simpleTypes/simpleTypes.js
```

## Practice
- Add a `city: string` variable and print it.
- Create `isRemote: boolean` and toggle it.
- Change `dynamicValue` to an object and narrow it safely.
